# 🚀 Redux Counter App

A modern and interactive counter application built with **React**, **Redux Toolkit**, and **Tailwind CSS**.

---

## ✨ Features

* 🔢 Increment / Decrement counter
* ➕ Increment by custom amount
* ➖ Decrement by custom amount with validation
* ⚡ Increment/Decrement by 5
* ✖️ Reset functionality
* ✨ Multiply counter value
* 🚫 Error handling with user feedback
* 🎨 Clean and modern UI design

---

## 🛠️ Tech Stack

* React
* Redux Toolkit
* Tailwind CSS

---

## 📸 UI Preview

Modern card-based UI with:

* Centered layout
* Gradient background
* Clean spacing & typography
* Interactive buttons
